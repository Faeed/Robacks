Start Receiving 40% cashback when you purchase something by linking a group game. Note that, you won't get cashback for limiteds/collectibles. To get started:
  1. Select a game which is under one of your groups or any group you have access to it's funds.
  2. Make the game public and accessible for anyone to play.
  3. Copy the game's ID from URL and paste it inside the "Enter Game ID.." textbox below.
  4. Press the "Link" button and you are ready to go!
If the tutorial didn't clear it up, or you have any questions - please check out the devforum post by pressing the DevForum logo at the bottom. There's a detailed tutorial, and a YouTube video tutorial within the post. If purchases don't work, please reinstall.
